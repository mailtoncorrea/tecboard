# 🚀 TecBoard

Projeto desenvolvido durante meus estudos de **Front-End** com a Alura.

O **TecBoard** é uma aplicação web criada com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end, como estruturação de layout e estilização com CSS.

---

## 📚 Sobre o Projeto

O TecBoard foi desenvolvido como parte do meu processo de aprendizado em:

- HTML5
- CSS3
- Estruturação de layout
- Responsividade
- Boas práticas de organização de código

Este projeto tem foco educacional e representa minha evolução prática na construção de interfaces web.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3

---

## 🎯 Objetivos de Aprendizado

- Criar layouts modernos e organizados
- Trabalhar com responsividade
- Melhorar a semântica do HTML
- Aplicar estilização com CSS
- Estruturar um projeto front-end do zero
